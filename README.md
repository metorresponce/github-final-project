# Simple Interest Calculator

## Description

This project contains a simple interest calculator that computes the interest earned on a principal amount over a specified period of time at a given annual interest rate.

The calculator uses the simple interest formula:

**Simple Interest = (Principal × Rate × Time) / 100**

## Features

- Calculates simple interest
- Accepts principal amount, annual interest rate, and time period as input
- Displays the calculated interest clearly

## How to Use

1. Enter the principal amount (initial investment).
2. Enter the annual interest rate (in percentage).
3. Enter the time period (in years).
4. The program calculates and displays the simple interest.

## Example

If:

- Principal = 1000  
- Rate = 5  
- Time = 2  

Then:

Simple Interest = (1000 × 5 × 2) / 100 = 100  

The interest earned is **100**.
